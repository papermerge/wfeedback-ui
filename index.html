LED Status Indicators

Attches LED status indicator to the elements with .led-status css class.

LED status can be for:

  • 1. Documents - in documents browser
  • 2. Pages - in document viewer

Documents

For Documents css classes must be .led-status.document

Pages

For Pages - css classes must be .led-status.page
